## 🎯 الشاشات الرئيسية الثلاث

### 1️⃣ لوحة تحكم الرواتب (Payroll Dashboard)
**المسار:** `/payroll/dashboard`  
**الأيقونة:** 💰 (أخضر)

#### الوظائف الرئيسية:
- ✅ عرض ملخص الرواتب الأسبوعية والشهرية
- ✅ فلترة البيانات حسب التاريخ والمجموعة
- ✅ عرض رسوم بيانية متقدمة:
  - توزيع الرواتب حسب المجموعة
  - معدل الحضور والانضباط
  - اتجاه الرواتب والموظفين
  - مؤشرات الأداء حسب المجموعة
- ✅ عرض جداول تفصيلية للرواتب اليومية

#### كيفية الاستخدام:

**الخطوة 1: اختيار التاريخ والمجموعة**
```
1. انقر على حقل "التاريخ" واختر التاريخ المطلوب
2. انقر على قائمة "المجموعة" واختر المجموعة (اختياري)
3. سيتم تحديث البيانات تلقائياً
```

**الخطوة 2: مراجعة الملخصات**
```
- إجمالي الرواتب: المبلغ الإجمالي للفترة المختارة
- إجمالي الموظفين: عدد الموظفين الذين بصموا
- إجمالي الساعات: عدد الساعات المعمول بها
```

**الخطوة 3: تحليل الرسوم البيانية**
```
- استخدم الرسوم البيانية لفهم توزيع الرواتب والحضور
- قارن أداء المجموعات المختلفة
- تابع الاتجاهات على مدار الأسبوع
```

#### المنطق الحسابي:
```sql
-- الراتب اليومي يُحسب بناءً على:
1. الساعات الفعلية المعمول بها
2. معدل الراتب بالساعة للموظف
3. البدلات والخصومات (إن وجدت)
4. التأخير والانصراف المبكر
5. البصمات الناقصة (معالجة تلقائية)
```

### 2️⃣ الورديات الديناميكية (Dynamic Schedules)
**المسار:** `/schedules/dynamic`  
**الأيقونة:** ⏰ (أزرق)

#### الوظائف الرئيسية:
- ✅ عرض جداول العمل لكل مجموعة
- ✅ تعديل ساعات العمل (الخميس، الجمعة، وبقية الأسبوع)
- ✅ تعديل الساعات المطلوبة يومياً
- ✅ تفعيل/تعطيل الجداول الزمنية

#### كيفية الاستخدام:

**الخطوة 1: اختيار المجموعة**
```
1. انقر على قائمة اختيار المجموعة
2. اختر المجموعة المطلوبة (الفنيين، الإداريين، الأمن، الصيانة)
3. سيتم عرض الجداول الزمنية للمجموعة
```

**الخطوة 2: تعديل الجدول الزمني**
```
1. انقر على زر "تعديل" بجانب اليوم المطلوب
2. عدّل ساعات البداية والنهاية
3. عدّل عدد الساعات المطلوبة
4. انقر "حفظ" لتطبيق التغييرات
```

**الخطوة 3: تفعيل/تعطيل**
```
1. استخدم الزر "تفعيل/تعطيل" لتفعيل أو تعطيل جدول معين
2. الجداول المعطلة لن تؤثر على حسابات الرواتب
```

#### ملاحظات مهمة:
```
⚠️ التغييرات تُطبق فوراً على قاعدة البيانات
⚠️ تأكد من صحة الساعات قبل الحفظ
⚠️ الجداول المعدلة تؤثر على حسابات الرواتب المستقبلية
```

### 3️⃣ مركز مراجعة البصمات (Punches Review Center)
**المسار:** `/punches/review`  
**الأيقونة:** ✓ (برتقالي) مع Badge عدد التنبيهات

#### الوظائف الرئيسية:
- ✅ عرض البصمات المعلقة للمراجعة
- ✅ عرض البصمات المكتملة تلقائياً
- ✅ الموافقة أو رفض البصمات
- ✅ إضافة ملاحظات على كل بصمة
- ✅ فلترة حسب الحالة والموظف والمجموعة

#### كيفية الاستخدام:

**الخطوة 1: عرض البصمات المعلقة**
```
1. انتقل إلى مركز مراجعة البصمات
2. ستظهر قائمة بالبصمات التي تحتاج مراجعة
3. Badge بجانب الأيقونة يعرض عدد البصمات المعلقة
```

**الخطوة 2: مراجعة البصمة**
```
1. انقر على البصمة لفتح تفاصيلها
2. ستظهر معلومات:
   - اسم الموظف والمجموعة
   - التاريخ والوقت
   - نوع المشكلة (بصمة ناقصة، وقت غير صحيح، إلخ)
   - الملاحظات السابقة (إن وجدت)
```

**الخطوة 3: اتخاذ القرار**
```
أ) الموافقة على البصمة:
   - انقر زر "موافقة"
   - أضف ملاحظة (اختياري)
   - انقر "تأكيد"
   
ب) رفض البصمة:
   - انقر زر "رفض"
   - أضف سبب الرفض (مطلوب)
   - انقر "تأكيد"
   
ج) تأجيل المراجعة:
   - انقر "تأجيل" للمراجعة لاحقاً
```

#### أنواع البصمات المعلقة:

| النوع | الوصف | الحل |
|-------|--------|------|
| **بصمة ناقصة (Check-In)** | الموظف لم يبصم الدخول | موافقة أو رفض |
| **بصمة ناقصة (Check-Out)** | الموظف لم يبصم الخروج | موافقة أو رفض |
| **وقت غير صحيح** | البصمة في وقت غير متوقع | تصحيح أو موافقة |
| **تأخير** | الموظف تأخر عن الموعد | موافقة مع ملاحظة |

## 🧮 العمليات الحسابية

### 1. حساب الراتب اليومي

```
الراتب اليومي = (الساعات الفعلية × معدل الراتب بالساعة) + البدلات - الخصومات

حيث:
- الساعات الفعلية = وقت الخروج - وقت الدخول
- معدل الراتب بالساعة = الراتب الشهري ÷ (عدد ساعات العمل الشهرية)
- البدلات = بدلات إضافية (إن وجدت)
- الخصومات = خصومات بسبب التأخير أو الغياب
```

### 2. كشف البصمات الناقصة

```
النظام يكتشف تلقائياً:
1. عدم وجود بصمة دخول (Check-In)
2. عدم وجود بصمة خروج (Check-Out)
3. فجوات زمنية غير عادية بين البصمات
4. بصمات في أوقات غير متوقعة

⚠️ البصمات الناقصة تُعلم المدير فوراً عبر Badge في اللوحة الجانبية
```

### 3. الإغلاق التلقائي للبصمات

```
النظام يُغلق البصمات تلقائياً:
1. في نهاية يوم العمل (الساعة 6 مساءً)
2. إذا كانت البصمة مفتوحة لأكثر من 12 ساعة
3. عند تأكيد المدير على اكتمال اليوم

⚠️ الإغلاق التلقائي يحفظ البيانات في قاعدة البيانات بشكل دائم
```

### 4. ملخص رواتب المجموعة

```
إجمالي رواتب المجموعة = مجموع الرواتب اليومية لجميع الموظفين

معدل الحضور = (عدد الموظفين الذين بصموا ÷ إجمالي الموظفين) × 100%

معدل الانضباط = (عدد الموظفين في الوقت ÷ إجمالي الموظفين) × 100%
```

### مستويات الوصول

| المستوى | الوصول | الشاشات المتاحة |

| **Admin (مسؤول)** | وصول كامل | جميع الشاشات والعمليات |

| **Employee (موظف)** | قراءة فقط | عرض بيانات شخصية فقط |

### Row Level Security (RLS)

```
النظام يستخدم RLS لضمان:
✅ كل مدير يرى فقط بيانات مجموعته
✅ كل موظف يرى فقط بيانته الشخصية
✅ المسؤول يرى جميع البيانات
✅ لا يمكن الوصول للبيانات الحساسة من المتصفح
✅ جميع العمليات محمية بالتشفير
```

### حماية البيانات

```
🔒 التشفير:
   - جميع البيانات مشفرة في الانتقال (SSL/TLS)
   - كلمات المرور مشفرة في قاعدة البيانات

🔒 المصادقة:
   - تسجيل دخول آمن عبر OAuth
   - جلسات آمنة مع انتهاء الصلاحية التلقائي

🔒 التدقيق:
   - جميع العمليات مسجلة في قاعدة البيانات
   - يمكن تتبع من قام بأي عملية ومتى
```

## 🔧 استكشاف الأخطاء

### المشاكل الشائعة والحلول

#### 1. البصمات لا تظهر في لوحة التحكم

**الأسباب المحتملة:**
- البيانات لم تُحقن بعد
- الفلاتر تخفي البيانات
- مشكلة في الاتصال بقاعدة البيانات

**الحل:**
```
1. تحقق من الفلاتر (التاريخ والمجموعة)
2. امسح الفلاتر وحاول مرة أخرى
3. تحديث الصفحة (F5)
4. إذا استمرت المشكلة، اتصل بالدعم الفني
```

#### 2. الرسوم البيانية فارغة

**الأسباب المحتملة:**
- لا توجد بيانات للفترة المختارة
- المجموعة المختارة لا تحتوي على بيانات
- مشكلة في تحميل البيانات

**الحل:**
```
1. اختر تاريخ به بيانات (مثلاً: 27-31 يناير 2026)
2. حاول بدون تحديد مجموعة معينة
3. تحديث الصفحة
4. إذا استمرت المشكلة، اتصل بالدعم الفني
```

#### 3. لا يمكن تعديل الجداول الزمنية

**الأسباب المحتملة:**
- عدم وجود صلاحيات كافية
- الجدول معطل أو محمي
- مشكلة في الاتصال بقاعدة البيانات

**الحل:**
```
1. تأكد من أنك مسؤول (Admin)
2. تحديث الصفحة
3. حاول في متصفح مختلف
4. إذا استمرت المشكلة، اتصل بالدعم الفني
```

#### 4. Badge عدد البصمات المعلقة لا يظهر

**الأسباب المحتملة:**
- لا توجد بصمات معلقة
- مشكلة في تحميل البيانات
- مشكلة في الاتصال بقاعدة البيانات

**الحل:**
```
1. تحديث الصفحة (F5)
2. امسح ذاكرة التخزين المؤقت (Ctrl+Shift+Delete)
3. حاول في متصفح مختلف
4. إذا استمرت المشكلة، اتصل بالدعم الفني
```
