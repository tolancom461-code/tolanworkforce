# TolanWorkforce - نظام إدارة القوى العاملة الشامل

## نظرة عامة

**TolanWorkforce** هو نظام متكامل لإدارة الموارد البشرية والقوى العاملة، مصمم خصيصاً للشركات والمؤسسات الكبيرة. يوفر النظام حلاً شاملاً لإدارة الموظفين والحضور والانصراف والرواتب والتقارير المالية.

## المميزات الرئيسية

### 1. **إدارة الموظفين والمجموعات**
- إدارة كاملة لبيانات الموظفين (الاسم، الصورة، رقم الهوية، إلخ)
- تنظيم الموظفين في مجموعات عمل
- نظام QR Code لكل موظف
- تحميل وتخزين الصور بكفاءة

### 2. **نظام الحضور والانصراف**
- تسجيل الحضور والانصراف عبر QR Code
- ماسح ضوئي للكود QR
- تسجيل يدوي كخيار بديل
- حساب ساعات العمل تلقائياً
- تقارير يومية وشهرية

### 3. **تصدير التقارير إلى Excel**
- تصدير تقارير الحضور والانصراف
- ملخصات إحصائية بصيغ متعددة:
  - تقرير مفصل لجميع السجلات
  - ملخص بالعامل
  - ملخص بالمجموعة
  - ملخص بمركز التكلفة
- معاملات تصفية متقدمة (التاريخ، المجموعة، مركز التكلفة)

### 4. **نظام الرواتب والمالية**
- حساب الرواتب تلقائياً بناءً على الحضور
- تطبيق الخصومات والإضافات
- نظام الاستثناءات والتجاوزات
- تقارير مالية شاملة:
  - تقرير مالي للعامل
  - تقرير مالي للمجموعة
  - تقرير مالي لمركز التكلفة
- دفعات رواتب مع سير عمل الموافقة

### 5. **نظام الأدوار والصلاحيات**
- 6 أدوار افتراضية:
  - **Super Admin**: صلاحيات كاملة
  - **Admin**: إدارة النظام
  - **HR Admin**: إدارة الموارد البشرية
  - **Accountant**: إدارة الحسابات
  - **Supervisor**: الإشراف على المجموعات
  - **Guard**: الحراس والمراقبة
- 44+ صلاحية محددة
- تحكم دقيق على المستوى الإجرائي

### 6. **لوحة التحكم التنفيذية**
- عرض ذكي للبيانات الهامة
- تحليل الحضور والغياب
- كشف الشذوذ والأنماط
- توقعات نهاية اليوم
- نقاط الضغط والمشاكل

### 7. **إدارة مراكز التكلفة والمجموعات**
- تنظيم الموظفين حسب مراكز التكلفة
- إدارة الورديات والساعات
- تحديد أيام العمل والعطل
- إعدادات مرنة لكل مجموعة

## البنية التقنية

### Frontend
- **React 19** مع TypeScript
- **Tailwind CSS 4** للتصميم
- **shadcn/ui** للمكونات
- **tRPC** للاتصال مع الخادم
- **Vite** كأداة البناء

### Backend
- **Express 4** كإطار العمل
- **Node.js** مع TypeScript
- **tRPC** لإدارة الإجراءات
- **Drizzle ORM** لإدارة قاعدة البيانات

### قاعدة البيانات
- **MySQL/TiDB** (عبر Supabase)
- **20 جدول** منظمة بعناية
- **Drizzle Kit** للهجرات

### الاختبارات
- **Vitest** لاختبارات الوحدة
- **150+ اختبار** شامل
- تغطية شاملة للوظائف الأساسية

## التثبيت والتشغيل

### 1. التثبيت
```bash
cd /home/ubuntu/tolanworkforce
pnpm install
```

### 2. إعداد قاعدة البيانات
```bash
pnpm db:push
```

### 3. تشغيل الخادم
```bash
pnpm dev
```

### 4. الوصول إلى التطبيق
افتح المتصفح على: `http://localhost:3000`
