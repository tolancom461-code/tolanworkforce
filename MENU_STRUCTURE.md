# هيكل قوائم النظام

تم إعادة تنظيم قوائم النظام بشكل احترافي حسب الوظائف والأدوار:

## 1. لوحات التحكم
- **لوحة التحكم الرئيسية** (`/dashboard`) - نظرة عامة على إحصائيات النظام
- **لوحة التحكم التنفيذية** (`/executive`) - تقارير تنفيذية للإدارة العليا

## 2. إدارة الموارد البشرية
- **العمال** (`/workers`) - إدارة بيانات العمال
- **المجموعات** (`/groups`) - تنظيم العمال في مجموعات
- **مراكز التكلفة** (`/cost-centers`) - إدارة مراكز التكلفة

## 3. نظام الحضور والانصراف
- **تسجيل الحضور** (`/attendance`) - تسجيل حضور وانصراف العمال
- **سجل الحضور** (`/attendance/log`) - عرض سجلات الحضور
- **تعديل الحضور** (`/attendance/adjust`) - تعديل سجلات الحضور
- **تقارير الحضور** (`/attendance/reports`) - تقارير تحليلية للحضور
- **إدارة أيام العمل** (`/work-days`) - تحديد أيام العمل والإجازات

## 4. النظام المالي
- **الخصومات والإضافات** (`/finance/entry`) - إدخال الخصومات والإضافات
- **الاستثناءات المالية** (`/finance/overrides`) - إدارة الاستثناءات المالية
- **دفعات الرواتب** (`/payroll/batches`) - إنشاء ومراجعة دفعات الرواتب
- **التقارير المالية** (`/finance/reports`) - تقارير مالية شاملة

## 5. إدارة النظام
- **المستخدمين** (`/users`) - إدارة حسابات المستخدمين
- **الأدوار والصلاحيات** (`/roles`) - إدارة الأدوار
- **إدارة الصلاحيات** (`/permissions`) - تخصيص الصلاحيات

---

## مميزات التنظيم الجديد:
1. **تصنيف منطقي** - القوائم مصنفة حسب الوظائف الرئيسية
2. **سهولة التنقل** - المستخدم يجد ما يحتاجه بسرعة
3. **قابلية التوسع** - يمكن إضافة قوائم جديدة بسهولة
4. **احترافية** - تنظيم يتماشى مع أفضل الممارسات

## الأدوار المستهدفة:
- **المدير التنفيذي**: لوحات التحكم + التقارير
- **مدير الموارد البشرية**: إدارة العمال + الحضور
- **المحاسب**: النظام المالي
- **المدير المالي**: التقارير المالية + الموافقات
- **مشرف الحضور**: نظام الحضور والانصراف
- **مدير النظام**: إدارة النظام
