## كيفية استخدام البيانات التجريبية

### تسجيل الدخول

يمكنك تسجيل الدخول بأي من المستخدمين المذكورين أعلاه لاختبار الصلاحيات المختلفة:

```
اسم المستخدم: superadmin
كلمة المرور: hashed_password_placeholder
```

### استكشاف النظام

1. **لوحة التحكم:** عرض الإحصائيات والتحليلات الذكية
2. **إدارة المستخدمين:** عرض وتعديل المستخدمين والصلاحيات
3. **إدارة المجموعات:** عرض المجموعات والورديات
4. **إدارة العمال:** عرض العمال وطباعة بطاقات QR
5. **تسجيل الحضور:** تسجيل حضور جديد أو مسح QR Code
6. **التقارير:** عرض تقارير الحضور والرواتب وتصديرها إلى Excel

### مسح QR Code

يمكنك طباعة بطاقة عامل من صفحة تفاصيل العامل، ثم استخدام كاميرا الجهاز لمسح الرمز في صفحة تسجيل الحضور.

## إعادة إدخال البيانات

إذا كنت بحاجة إلى إعادة إدخال البيانات التجريبية:

```bash
cd /home/ubuntu/tolanworkforce
npx tsx seed-demo-data.mjs
```

**ملاحظة:** سيقوم الـ script بحذف جميع البيانات التجريبية القديمة قبل إدخال البيانات الجديدة.
