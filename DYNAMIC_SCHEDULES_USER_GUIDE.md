## 🎯 الخطوات الأساسية

### الخطوة 1: فتح شاشة الورديات الديناميكية

```
1. من اللوحة الجانبية (Sidebar)
2. ابحث عن "إدارة الورديات الديناميكية"
3. انقر على الأيقونة الزرقاء (⏰)
4. ستُفتح الشاشة مباشرة
```

### الخطوة 2: اختيار المجموعة

```
1. ستظهر قائمة بجميع المجموعات الأربع
2. اختر المجموعة التي تريد تعديل جدولها
3. مثال: اختر "الفنيين" لتعديل جدول الفنيين
```

### الخطوة 3: عرض الجداول الزمنية

```
بعد اختيار المجموعة، ستظهر جداول العمل:

┌─────────────────────────────────────────────────┐
│ اليوم    │ البداية │ النهاية │ الساعات │ الحالة │
├─────────────────────────────────────────────────┤
│ السبت    │ 08:00  │ 16:00  │ 8      │ ✓     │
│ الأحد    │ 08:00  │ 16:00  │ 8      │ ✓     │
│ الاثنين  │ 08:00  │ 16:00  │ 8      │ ✓     │
│ الثلاثاء │ 08:00  │ 16:00  │ 8      │ ✓     │
│ الأربعاء │ 08:00  │ 16:00  │ 8      │ ✓     │
│ الخميس   │ 08:00  │ 16:00  │ 8      │ ✓     │
│ الجمعة   │ 00:00  │ 00:00  │ 0      │ ✗     │
└─────────────────────────────────────────────────┘
```

### الخطوة 4: تعديل الجدول الزمني

#### طريقة 1: تعديل ساعات العمل

```
1. انقر على حقل "البداية" (Start Time)
2. اختر الساعة والدقيقة الجديدة
3. انقر على حقل "النهاية" (End Time)
4. اختر الساعة والدقيقة الجديدة
5. سيتم حساب الساعات تلقائياً
6. انقر "حفظ" لتطبيق التغييرات
```

#### طريقة 2: تعديل عدد الساعات المطلوبة

```
1. انقر على حقل "الساعات المطلوبة" (Required Hours)
2. أدخل عدد الساعات الجديد
3. انقر "حفظ" لتطبيق التغييرات
```

#### طريقة 3: تفعيل/تعطيل اليوم

```
1. انقر على زر "تفعيل/تعطيل" (Toggle)
2. إذا كان اليوم مفعل (✓):
   - سيتم تعطيله (✗)
   - لن يؤثر على حسابات الرواتب
3. إذا كان اليوم معطل (✗):
   - سيتم تفعيله (✓)
   - سيؤثر على حسابات الرواتب
```

### الخطوة 5: حفظ التغييرات

```
1. بعد تعديل أي حقل، انقر "حفظ"
2. ستظهر رسالة تأكيد: "تم الحفظ بنجاح"
3. التغييرات تُطبق فوراً على قاعدة البيانات
4. ستؤثر على حسابات الرواتب المستقبلية
```

## 💡 أمثلة عملية

### مثال 1: تغيير ساعات العمل للفنيين

**الحالة الحالية:**
```
السبت: 08:00 - 16:00 (8 ساعات)
```

**المطلوب:**
تغيير ساعات العمل إلى 07:00 - 15:00 (8 ساعات)

**الخطوات:**
```
1. اختر مجموعة "الفنيين"
2. انقر على حقل "البداية" لليوم السبت
3. اختر 07:00
4. انقر على حقل "النهاية"
5. اختر 15:00
6. انقر "حفظ"
✅ تم التغيير بنجاح!
```

### مثال 2: إضافة ساعات عمل إضافية يوم الخميس

**الحالة الحالية:**
```
الخميس: 08:00 - 16:00 (8 ساعات)
```

**المطلوب:**
إضافة ساعة إضافية (9 ساعات)

**الخطوات:**
```
1. اختر مجموعة "الإداريين"
2. انقر على حقل "النهاية" لليوم الخميس
3. اختر 17:00 (بدلاً من 16:00)
4. انقر "حفظ"
✅ تم التغيير بنجاح! (الآن 9 ساعات)
```

### مثال 3: تعطيل يوم الجمعة

**الحالة الحالية:**
```
الجمعة: معطل (✗) - 0 ساعات
```

**المطلوب:**
تفعيل يوم الجمعة كيوم عمل

**الخطوات:**
```
1. اختر مجموعة "الأمن"
2. انقر على زر "تفعيل" لليوم الجمعة
3. اختر ساعات العمل (مثلاً: 08:00 - 14:00)
4. انقر "حفظ"
✅ تم تفعيل يوم الجمعة! (الآن 6 ساعات)
```

### مثال 4: تغيير جدول مجموعة كاملة

**الحالة الحالية:**
```
مجموعة الصيانة:
- السبت إلى الخميس: 08:00 - 16:00 (8 ساعات)
- الجمعة: معطل
```

**المطلوب:**
تغيير الجدول إلى:
- السبت إلى الأربعاء: 07:00 - 15:00 (8 ساعات)
- الخميس: 07:00 - 14:00 (7 ساعات)
- الجمعة: معطل

**الخطوات:**
```
1. اختر مجموعة "الصيانة"
2. عدّل السبت: 07:00 - 15:00 → انقر "حفظ"
3. عدّل الأحد: 07:00 - 15:00 → انقر "حفظ"
4. عدّل الاثنين: 07:00 - 15:00 → انقر "حفظ"
5. عدّل الثلاثاء: 07:00 - 15:00 → انقر "حفظ"
6. عدّل الأربعاء: 07:00 - 15:00 → انقر "حفظ"
7. عدّل الخميس: 07:00 - 14:00 → انقر "حفظ"
✅ تم تغيير جدول المجموعة بالكامل!
```

## ⚠️ الأخطاء الشائعة

### الخطأ 1: عدم ظهور الجداول الزمنية

**السبب:**
- لم تختر مجموعة بعد
- مشكلة في الاتصال بقاعدة البيانات

**الحل:**
```
1. تأكد من اختيار مجموعة من القائمة
2. تحديث الصفحة (F5)
3. حاول في متصفح مختلف
```

### الخطأ 2: التغييرات لا تُحفظ

**السبب:**
- عدم الضغط على زر "حفظ"
- مشكلة في الاتصال بقاعدة البيانات
- انتهاء جلسة المستخدم

**الحل:**
```
1. تأكد من الضغط على "حفظ" بعد كل تعديل
2. انتظر رسالة التأكيد "تم الحفظ بنجاح"
3. تحديث الصفحة
4. إذا استمرت المشكلة، سجّل الدخول مرة أخرى
```

### الخطأ 3: ساعات غير صحيحة

**السبب:**
- اختيار ساعات غير منطقية (مثلاً: النهاية قبل البداية)
- خطأ في الحساب التلقائي

**الحل:**
```
1. تأكد من أن وقت النهاية > وقت البداية
2. مثال صحيح: 08:00 - 16:00 ✓
3. مثال خاطئ: 16:00 - 08:00 ✗
4. إذا حدث خطأ، عدّل مرة أخرى
```

### الخطأ 4: عدم ظهور التغييرات في لوحة الرواتب

**السبب:**
- التغييرات لم تُطبق بعد على الرواتب
- قد تحتاج إلى تحديث الصفحة

**الحل:**
```
1. تحديث صفحة لوحة الرواتب (F5)
2. اختر نفس المجموعة والتاريخ
3. ستظهر الرواتب المحدثة
```

## 💡 نصائح مهمة

### نصيحة 1: التخطيط المسبق

```
قبل البدء في التعديلات:
✓ اجمع متطلبات جميع المجموعات
✓ تأكد من صحة الساعات
✓ خطط للتغييرات بعناية
✓ ابدأ بمجموعة واحدة
```

### نصيحة 2: الحفظ الدوري

```
أثناء التعديلات:
✓ احفظ بعد كل تعديل
✓ انتظر رسالة التأكيد
✓ لا تغلق الصفحة قبل الحفظ
✓ استخدم الزر "حفظ" وليس الإغلاق المباشر
```

### نصيحة 3: المراجعة قبل التطبيق

```
قبل تطبيق التغييرات:
✓ راجع جميع الساعات
✓ تأكد من صحة الأيام
✓ تحقق من التأثير على الرواتب
✓ استشر الإدارة إذا لزم الأمر
```

### نصيحة 4: التوثيق

```
احتفظ بسجل للتغييرات:
✓ اكتب التاريخ والمجموعة
✓ اكتب الساعات القديمة والجديدة
✓ اكتب السبب وراء التغيير
✓ احفظ النسخة الاحتياطية
```

### نصيحة 5: التواصل

```
بعد التعديلات:
✓ أخبر الموظفين عن التغييرات
✓ أرسل نسخة من الجدول الجديد
✓ اطلب التأكيد من المديرين
✓ وثّق التغييرات في السجلات
```
